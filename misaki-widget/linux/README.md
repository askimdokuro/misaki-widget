If you are not satisfied with a setting, you can change it via config.ini


width: determines the width (the height is calculated automatically based on the aspect ratio)

x_position/y_position: determines the widget's position in pixels

panel_height: determines the distance (in centimeters) from the bottom of the screen

text_color: changes the text color (uses HTML color codes)

debug: setting this to 1 displays the pink box hitbox (the area containing the date/time); setting it to 0 hides it

box_x/box_y: determines the position of the pink box hitbox

box_width/box_height: determines the dimensions of the pink box hitbox
